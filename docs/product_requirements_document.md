# Schedule Reminder Extension - 产品需求文档 (PRD)

## 1. 文档概述

### 1.1 产品简介
**Schedule Reminder** 是一款功能完善的 Chrome 浏览器扩展，旨在帮助用户高效管理个人提醒。它提供精准的倒计时显示、灵活的重复提醒设置以及可靠的弹窗通知机制，确保用户不错过任何重要事项。

### 1.2 目标用户
需要轻量级时间管理工具的 Chrome 浏览器用户，特别是需要频繁设置定时提醒的办公人员和学生。

## 2. 核心功能需求

### 2.1 提醒管理
- **添加提醒**：
  - 输入标题（必填）。
  - 选择具体日期和时间（精确到分钟）。
  - 选择重复模式：不重复、每天、每周、每月。
  - 添加备注信息（选填）。
  - **交互优化**：表单底部使用 "Confirm" 按钮明确提交意图。
- **查看提醒**：
  - 列表展示所有提醒事项。
  - 实时显示倒计时（格式：天/时/分/秒），每秒刷新。
  - 显示提醒的重复状态（如 "Daily" 标签）。
  - 区分启用/禁用状态（禁用项显示删除线）。
- **编辑提醒**：
  - 点击列表项的“编辑”图标（铅笔）进入编辑模式。
  - 自动填充当前信息。
  - 支持修改所有字段。
  - 修改时间后自动重置倒计时和后台闹钟。
- **删除/禁用**：
  - **删除**：点击垃圾桶图标永久删除提醒。
  - **禁用/启用**：点击铃铛图标切换状态，不删除数据。

### 2.2 通知系统（双重保障）
- **系统通知**：
  - 当倒计时结束时，通过 Chrome 系统通知中心推送消息。
  - 显示标题和备注内容。
  - 设置 `requireInteraction: true`，确保通知不会自动消失。
- **独立弹窗 (Popup Window)**：
  - **强力提醒**：除了系统通知外，自动弹出一个独立的浏览器窗口。
  - **尺寸**：小巧精致的卡片式设计 (250x300px)。
  - **内容**：醒目的红色图标、事件标题、备注详情。
  - **操作**：
    - **Dismiss**：关闭弹窗。
    - **Edit (Reschedule)**：在弹窗内直接修改时间（如推迟 10 分钟），支持时间有效性验证（不能设置过去的时间）。

### 2.3 数据持久化与离线支持
- 使用 `chrome.storage.local` 存储所有提醒数据，确保浏览器重启后数据不丢失。
- 使用 `chrome.alarms` API 调度定时任务，支持浏览器后台运行。
- 完全支持离线工作，无需网络连接。

## 3. 用户界面 (UI) 规范

### 3.1 主界面 (Popup)
- **尺寸**：固定宽度 400px，高度自适应（最小 400px，最大 600px）。
- **风格**：现代简约风格，使用 Tailwind CSS 构建。
- **配色**：
  - 主色调：蓝色 (Blue-600) 用于主要按钮和强调文字。
  - 警告色：红色 (Red-600) 用于删除和过期提示。
  - 背景色：浅灰 (Gray-100) 用于页面背景，白色用于卡片。
- **图标**：统一使用 `Lucide React` 图标库。

### 3.2 提醒弹窗 (Reminder Window)
- **尺寸**：250x300px。
- **布局**：居中布局，强调视觉冲击力。
- **交互**：提供编辑模式，原地修改时间并保存。

## 4. 非功能需求

- **性能**：列表加载无延迟，倒计时流畅无卡顿。
- **兼容性**：支持 Chrome 最新版本（Manifest V3）。
- **安全性**：不收集用户隐私数据，所有数据仅存储在本地。
- **国际化**：当前版本仅支持英文界面（后续可扩展）。
